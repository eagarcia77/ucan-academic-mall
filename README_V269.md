# UCAN Academic Mall V269

## Corrección Meta Quest

- El joystick izquierdo mueve únicamente sobre los ejes X/Z.
- Se eliminó el movimiento vertical causado por la inclinación del visor.
- El joystick derecho aplica giros de confort de 30 grados.
- La cámara conserva la altura física del usuario y sigue el nivel de las escaleras.
- Se añadieron límites de velocidad, aceleración, posición y recuperación automática.
- El avatar local se oculta en VR y la cámara de escritorio se sincroniza para presencia compartida.

## Despliegue

Después de subir los cambios a GitHub, en Render seleccione **Manual Deploy → Deploy latest commit**.
