# UCAN Academic Mall V270

- Movimiento natural en Meta Quest con tres velocidades.
- Giro suave o por pasos de 30 grados.
- Bloqueo de paridad visual para conservar cielo, paisaje, iluminación, niebla y escala de render al entrar en VR.
- Limpieza de parámetros `avatar=1` y `password=change` para que no vuelvan a abrir el perfil después de completarlo.
- Migración de perfiles heredados.
- UCAN Avatar Studio con presets, paletas visuales, generación aleatoria y avatar 3D modernizado.
