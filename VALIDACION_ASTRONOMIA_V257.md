# Validación astronómica V264

- Mapa celeste polar orientado con norte, este, sur y oeste.
- Posición de estrellas calculada mediante tiempo sideral local para San Germán.
- Posiciones planetarias mediante efemérides educativas de baja precisión.
- Fase lunar dibujada gráficamente con terminador iluminado.
- Calendario mensual de eventos 2026 con eclipses y lluvias de meteoros de NASA.
- Aproximaciones de cometas consultadas mediante el API NASA/JPL SBDB a través del servidor local.
- Los objetos del cielo solo se muestran de noche, sobre el horizonte y con nubosidad aceptable.
