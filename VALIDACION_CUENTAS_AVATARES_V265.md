# Validación de cuentas y avatares V265

- Registro e inicio de sesión persistentes.
- Contraseñas derivadas con `crypto.scrypt` y sal individual.
- Sesiones mediante cookie `HttpOnly`, `SameSite=Lax` y `Secure` bajo HTTPS.
- Roles `admin` y `user`.
- Administración de cuentas, estado y restablecimiento de contraseña.
- Personalización de cabello, ropa, zapatos, tono de piel y accesorios.
- Vista en tercera persona mediante `FollowCamera`.
- Presencia de usuarios y avatares remotos.
- Preparación para GitHub Codespaces, GitHub Actions y GHCR.
